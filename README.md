# PDF Merger Tool 📄

Prosta aplikacja webowa zbudowana w **Flask** która pozwala na przesyłanie plików PDF i łączenie ich w jeden dokument

## Funkcje
* Przesyłanie wielu plików PDF jednoczesnie.
* Automatyczne łączenie plików w jeden dokument: `merged.pdf`.
* Natychmiastowe pobieranie gotowego pliku

## Technologie
* **Python 3.x**
* **Flask** (Framework webowy)
* **PyPDF2** (Biblioteka do manipulacji plikami PDF)
